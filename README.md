# Student-Management-System
A simple Student Management System built with Python to manage student records efficiently. This project allows users to add, view, search, update, and delete student information through a user-friendly console interface.
Features
Add new student records
View all students
Search students by ID or name
Update student details
Delete student records
Store and manage student data efficiently
Technologies Used
Python
Object-Oriented Programming (OOP)
File Handling (optional)
Purpose
This project was created to practice Python programming concepts such as functions, loops, conditionals, data structures, and OOP while building a real-world application.

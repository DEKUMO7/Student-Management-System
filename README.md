# 🎓 Student Management System (Python)

A simple Student Management System built with Python. This project stores student records using either a TXT file or a CSV file and allows users to manage student information through the terminal.

## ✨ Features

- ➕ Add Student
- 🔍 Search Student by Roll Number
- ✏️ Update Student Details
- ❌ Delete Student
- 💾 Store data in TXT or CSV (OPEN THE FILE IN EXEL , GOOGLE SHET AND etc.)
- 📋 Automatically creates the CSV header if it doesn't exist

### 2. Open the project folder

cd Student-Management-System
### 3. Run the TXT version

python "Add student (TXT file version).py"
### 4. Run the CSV version

python "student(CSV file version).py"
---

## 📖 How to Use

When the program starts, you will see:

ADD / SEARCH / DELETE / UPDATE / EXIT:
### ➕ ADD
Adds a new student record.

### 🔍 SEARCH
Searches for a student using the Roll Number.

### ✏️ UPDATE
Updates an existing student's information.

### ❌ DELETE
Deletes a student record using the Roll Number.

### 🚪 EXIT
Closes the program.
